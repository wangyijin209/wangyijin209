在手机上开mc服务器（java）step1

2021-10-04 mc linux 评论

准备工作

一台手机，一个樱花frp账号
软件：teemux or utermux or zerotermux

开始操作

你需要选择一个linux发行版（建议debian），新手建议用moe，不用安装图形

安装时一路回车，到最后安装图形时按CTRL-C退出

选择版本

浏览器打开https://mcversions.net/
选择你想玩的版本

点击download
右键download server jar，复制链接

进入termux

安装java（不会的出门右拐！）

别别别，开个玩笑，接着往下看

1.如果你选择的版本是1.17及以上，请选择java16
2.如果你选择的版本是1.17一下，可以选用java8或11
安装java方法
java8：https://bell-sw.com/pages/downloads/#/java-8-lts（选择linux的arm版本deb包）
java16：https://www.oracle.com/java/technologies/javase-jdk16-downloads.html
操作同上

开始运行jar

接下来会有一小段报错，是正常现象

你会看到多了两个文件

接下来编辑这两个文件

键入i，开始插入
将文档末尾的false改为true
按esc输入:wq!（注意是英文的:）

然后修改server.properties文件
还是用vi

键入i，插入
关闭出生点保护，你也可以不关:
将spawn-protection=16改为spawn-protection=0
将正版验证关闭（如果你和你的朋友都有正版mc，那当我没说）:
将online-mode=true改为online-mode=false
按esc，键入:wq!

这时已经基本配置好了，接下来启动服务器

1024也可以改大，看手机配置吧QAQ

step1到这里就结束了，如果有好朋友在局域网内就可以快乐的联机了。
